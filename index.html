O Projeto

O objetivo desse projeto é desenvolver um sistema integrado que seja capaz de administrar os investimentos de valores mensais e empréstimos em um fundo de reserva cooperativo. O sistema deve ser capaz de controlar os valores acumulados mensalmente, bem como os empréstimos concedidos, pagamentos, redistribuição de valores, rendimento acumulado no período, participantes e distribuição de valores ao final do período. O aplicativo apresentado abaixo é o produto final destinado aos participantes.

Splash

Ao executar o aplicativo, valida se o usuário já está autenticado e se o token de acesso ao aplicativo ainda está válido. A API usa autenticação por JWT que expira em 24 horas após o login.

Login

Autentica o usuário na API do aplicativo e fornece o token que será utilizado em cada requisição para a API. Este token contém informações sobre a credencial do usuário para que sejam utilizadas nas requisições para identificação e recuperação dos dados referentes a ele.

Dashboard

Tela inicial do aplicativo. Nela são exibidas as informações mais relevantes para acompanhamento do investimento do participante. Valor total acumulado pelo participante até o momento. Rendimento parcial do investimento considerando todos os participantes e suas movimentações e as últimas transações realizadas pelo participante

Mensalidades

Lista de todas as mensalidades do perído com as informações das mensalidades já pagas. Informações como juros, data de pagamento e valor total pago também são exibidas nessa tela para um acompanhamento mais detalhado do andamento do investimento do participante

Empréstimos

Lista de eventuais empréstimos contraídos pelo participante durante o perído. Nesta primeira visualização as informações são mais simplificadas dando uma ideia geral sobre a saúde financeira do participante. Ao tocar em um empréstimo, uma visão detalhada será exibida

Detalhamento de Empréstimo

Exibe um extrato detalhado de toda a movimentação do empréstimo. Datas de pagamento, saldos e toda informação necessária para que o participante possa acompanhar sem nenhum tipo de dúvida o histórico de pagamento de um empréstimo.

Regras do Fundo de Investimento

Esta visão detalha todo o funcionamento do fundo de investimento com as suas principais regras. É uma forma de auxiliar a retirada de dúvida de maneira simples e eficiente. Como em todo período novos participantes aderem ao fundo, é possível entender todo o funcionamento pelo próprio aplicativo.

Versões Futuras

Algumas melhorias já estão em desenvolvimento para tornar o aplicativo ainda mais completo e minimizar o máximo possível a necessidade de interações pessoais com os administradores do fundo, permitindo assim que possam se concentrar em outras atividades mais prioritárias para a gestão do fundo de investimento.


Solicitações de Empréstimo

Permitir que participantes possam solicitar um empréstimo diretamente pelo aplicativo.

Envio de Comprovantes de Pagamento

Permitir anexar comprovantes de pagamento para as mensalidades e empréstimos. Hoje esse recurso é feito via whatsapp e esses comprovantes acabam se perdendo com o tempo.

Notificações

Notificação de vencimento nos próximos dias para mensalidades e empréstimos

Aplicativo de Gestão

Criar um segundo aplicativo que será utilizado pelos gestores do fundo possibilitando ações como aprovações, envio de notificações, etc.

Sistema Web

Sistema para gerenciamento de todas as informações do fundo de investimento.

Consolidação Versão iOS

Homologar a versão para iOS e posteriormente publicar na loja.
